# Full desktop suite implementation status

This file distinguishes implemented software from hardware qualification. A UI control or adapter skeleton is not evidence that a device is safe to write.

`docs/qualification/reference-system.json` is a privacy-preserving one-system ledger and `pchelper-cli qualification --ledger ...` evaluates the published 18-system gate. It is intentionally blocked until signed physical-system records cover every required CPU/GPU/platform family and each claimed controller has two independent successful reports.

`0.5.5-alpha` is a source-level development line, not a Fan Control, Afterburner, or SignalRGB replacement claim. Composite GPU tuning, fan control, and exact-route RGB paths remain Experimental unless the public ledger records the required physical evidence. Restart, emergency, suspend, reboot, uninstall, signed-package, clean-machine, and long-soak gates remain open. `1.0` remains blocked by the published physical matrix.
